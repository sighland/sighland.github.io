Libraries

  • numpy

    • Concepts

    The term broadcasting refers to how numpy treats arrays of different shapes during arithmetic operations. The smaller array is "broadcast" across the larger array. We just need to reshape the arrays so that we can broadcast the value across the larger array.
    To dimensions are compatible if either of them are equal, or one of them is one. Thus, input arrays don't need to have the same number of dimensions. If a dimension has size 1, we copy the value from the other array.
