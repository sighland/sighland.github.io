Least Squares

When taking measurements, there's almost always error. To smooth out this error, we take multiple measurements. We then take the average of these measurements: so we are using a single number to represent a whole set of numbers. We seek a lower-dimensional approx to a higher-dimensional object -- this involves projecting a vector onto a lower dimensional subspace, which is what we do through least squares.

Terms

Overdetermined - a system where there are more equations that unknowns (long and skinny matrix) Residual - for a system Ax=b, the difference between the LHS and RHS (r = b-Ax)
