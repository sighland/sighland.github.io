Overview

There are several types of error in the approximations used in numerical analysis:

  • Computational Error - difference between exact and approximate functions for same input
    • Truncation Error - difference between a true result and the result given by a function
    • Rounding Error - difference between result given from a function using true arithmetic, and result given using rounded inputs
  • Data Error - difference between exact function values due to error in input value(s)
  • Absolute Error - difference between the approx value and the true value
  • Relative Error - absolute error divided by the true value (can be expressed as percentage when multiplied by 100)

Additionally, problems can be classified as insensitive (well-conditioned) or sensitive (ill-conditioned): that is to say, for a sensitive problem, a small relative change in the input data would lead to a large relative change in the output of the function.
