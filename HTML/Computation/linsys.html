Linear Systems

Nonlinear systems can oftentimes be approximated locally by linear systems. That is, within a certain range of a value, a linear system gives us a pretty good estimate as to the solution of a nonlinear system. For instance, a derivative is just a linear approximation of a nonlinear curve at a particular point. Given the equation Ax = b, if there exists a vector b that can be expressed as a linear combo of the columns of matrix A, then the system is consistent, and these coefficients of the lin combo are given by vector x.
The simplest of linear systems to solve are square systems, where # columns = # rows in the A matrix.


If we want to consider sensitivity of a system, we need to measure perturbations of the solution x to the input data. We need some notion of size to do this, which is where vector norms come into play. There are 3 types of p-norms:
  1. 1-norm
  2. 2-norm
  3. ∞-norm

The 2-norm is typically called the euclidean norm, and it's most commonly used to find the magnitude of a vector. On the other hand, the other two p-norms are used mostly for determining sensitivity of sols to lin systems. The norm of a vector can also be thought of as the factor by which the unit sphere must be scaled to encompass the vector.

Likewise, we need to measure the size of a matrix using matrix norms.
The matrix norm corresponding to the vector 1-norm is just the max absolute column sum of the matrix.
The matrix norm corresponding to the vector ∞-norm is just the max absolute row sum.

Matrices also have a condition number consistent with the general notion of a condition number. That is, the cond number bounds the ratio of relative change in the sol of a lin sys to a certain relative change in the input data. The condition number is the ratio of maximum relative stretching to the max relative shrinking. A large number indicates a transformation that is longer and thinner. There are 4 important properties of the matrix condition number:
  • For any matrix A, cond(A)≥1
  • For the identity matrix, cond(I)=1
  • For any matrix A and nonzero scalar ɣ, cond(ɣA)=cond(A)
  • For any diagonal matrix D = diag(di), cond(D) = (max|di|) / (min|di|)

The condition number is also a measure of how close a matrix is to being singular - a larger cond # means closer to being singular.
