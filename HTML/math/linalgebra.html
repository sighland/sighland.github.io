Overview

Linear Algebra is an essential tool for solving systems of equations. Specifically, linear systems.

Topics
  • The Eigenvalue Problem

Terms

  • Row equivalent - one matrix can be transformed into the other matrix by a sequence of elementary row operations (replacement, interchange, or scaling)
  • Echelon Form - a matrix form that satisfies the following criteria:
    1. All nonzero roes are above any rows with all zeroes
    2. The leading entry (first nonzero from left) of a nonzero row is always strictly to the right of the leading entry of the row above it
  • Row Reduced Echelon Form (RREF) - a matrix is in echelon form and satisfies:
    1. The leading entry in each nonzero row is 1
    2. Each leading entry is the only nonzero entry in its column
  • Pivot position - the position of a leading entry in an echelon form of a matrix
  • Pivot column - the column that contains a pivot position
  • Pivot variable / basic variable - a variable that corresponds to a pivot column in the coefficient matrix of a linear system
  • Free variable - variable that is not a pivot variable
  • Consistent System - a system that has a solution
  • Span - given a set of vectors S, a set of all lin combos of all vectors in S
  • Rank - the number of linearly independent solutions

Subspaces

In matrix theory, there are four fundamental subspaces:

  1. Column Space
  2. Row Space
  3. Null Space
  4. Left Null Space

All vectors that are orthogonal to the row space are in the null space, and vice versa. So, a given vector x can be decomposed into xr + xn = x, meaning that it has a component in the null space and a component in the row space. The xn component maps to the 0 vector in the column space. Similarly, if we have Ax = b, then xr maps to b in the column space.

Much like the row and null spaces are orthogonal to each other, the column and left null spaces are orthogonal to each other.

Existence and Uniqueness

A square n x n matrix, A is said to be nonsingular if it satisfies any of the following criteria:

  1. A has an inverse
  2. det(A) does not equal 0
  3. rank(A) = n (the rank of the matrix is the maximum number possible)
  4. For any nonzero vector x, Ax does not equal 0

otherwise, the matrix is singular.

The singularity of a matrix is important in determining whether or not it is unique; if A if nonsingular, then it must have an inverse, and the system Ax = b always has a unique solution.

Similarly, if A is singular, then the number of solutions is determined by the RHS vector b. That is, for a given val of b, there may be no solution, but if there is a solution, then any additive scalar multiplied by a nonzero vector to the solution vector x is also a solution; moreover, this vector multiplying the scalar, z, must have the condition Az = 0, by condition 4 listed above.
Thus:
  • Unique Solution: A singular, b arbitrary
  • Infinitely Many Solutions: A singular, b within span(A) (consistent)
  • No Solution: A singular, b not within span(A) (inconsistent)
