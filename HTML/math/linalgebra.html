Existence and Uniqueness

A square matrix, A is said to be nonsingular if it satisfies any of the following criteria:

  1. A has an inverse
  2. det(A) does not equal 0
  3. rank(A) = n
  4. For any nonzero vector x, Ax does not equal 0

otherwise, the matrix is singular.

The singularity of a matrix is important in determining whether or not it is unique; if A if nonsingular, then it must have an inverse, and the system Ax = b always has a unique solution.

Similarly, if A is singular, then the number of solutions is determined by the RHS of vector b. That is, for a given val of b, there may be no solution, but if there is a solution, then any additive scalar multiplied by a nonzero vector to the solution vector x is also a solution; moreover, this vector multiplying the scalar, z, must have the condition Az = 0, by condition 4 listed above.
Thus:
  • Unique solution: A singular, b arbitrary
  • Infinitely many solutions: A singular, b within span(A) (consistent)
  • No solution: A singular, b not within span(A) (inconsistent)
