Overview

This syllabus assumes the user is working with the C language and using a Linux machine, though Windows (and probably IOS) is close enough that most things will still apply, with the exception of POSIX specifics. Still, if you are not using a Unix system, learn the POSIX specific concepts anyways. I recommend using Vim to program.
