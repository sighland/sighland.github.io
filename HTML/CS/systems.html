Overview

Systems programming is, at a basic level, the programming of dynamic computer systems. What does this mean? Well, it helps to consider the definitions
... by definitions. So: "programming" means coding. Dynamic means "actively changing"; i.e. we are working in the realm where we are actively recieving and
writing data between computational units. "System" just refers to the fact that what we are coding is part of a network of items -- for example, an operating
system is a network of operations.

On the Hardware - Software spectrum, this would count as software, but about one step removed from hardware.
